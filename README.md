# vite React TypeScript tailwindcss starter

Template for vite, React + tailwindcss + TypeScript projects with some tools preconfigured.

## About

Template for vite, React + tailwindcss + TypeScript projects with some tools preconfigured.

### Libraries

- [TypeScript](https://www.typescriptlang.org/)
- [vite](https://vitejs.dev/)
